## GPT Image 2 を使う理由
- 現実世界の知識が強く、物体や場面の表現がより正確
- スタイル理解が向上し、特定の美術様式やゲーム美学を再現しやすい
- 複雑な指示への追従性が高い
- 写真のように自然な出力を得やすい
