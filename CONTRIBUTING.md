# Contributing to Awesome GPT Image 2

First off, thank you for considering contributing! It's people like you that make this resource valuable for the community.

## How Can I Contribute?

### Adding New Prompts

We welcome new prompt submissions! When adding a prompt, please:

1. **Test your prompt** - Ensure it produces consistent, high-quality results
2. **Include the full prompt text** - Copy the exact prompt used
3. **Provide source attribution** - Link to the original X/Twitter post if applicable
4. **Add example images** - Include image URLs or upload to the `assets/` folder
5. **Categorize appropriately** - Place your prompt in the most relevant section

### Format for New Entries

```markdown
### [Descriptive Title]

![Example Image](image-url)

**Prompt:**
```
[your full prompt text here]
```

**Source:** [@username](https://x.com/username/status/123456789)

**Tips:** [Optional - any additional tips for using this prompt]
```

### Suggesting Categories

If you notice a gap in our categorization, feel free to suggest new sections that would help organize prompts better.

### Reporting Issues

Found a broken link? Prompt not working as expected? Please open an issue!

## Pull Request Process

1. Update the README.md with details of changes if needed
2. Ensure your contribution follows the existing format
3. The PR will be merged once reviewed

## Community Guidelines

- Be respectful and inclusive
- Give proper credit to original prompt creators
- Help others understand how to use prompts effectively

Thank you for helping make Awesome GPT Image 2 better! 🎨
